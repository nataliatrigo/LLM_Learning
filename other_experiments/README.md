# Other Experiments

These folders contain extensions and robustness exercises related to the main
finite-horizon model:

- [`average_cost/`](average_cost/): finite-state approximations to the
  average-reward problem.
- [`finite_memory/`](finite_memory/): finite-horizon learning with bounded
  memory.

The primary model, paper, and presentation-ready figures are in
[`../finite_horizon/`](../finite_horizon/). Local generated-data archives are
excluded from Git.
